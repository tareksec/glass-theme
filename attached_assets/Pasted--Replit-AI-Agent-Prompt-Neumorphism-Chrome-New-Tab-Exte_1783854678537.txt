## 1. Design System & Global Styles

- **Theme:** Neumorphism / Soft UI
- **Background:** `#e0e5ec` — use for body and all component surfaces
- **Text:** Deep grey/charcoal (`#3a3a4c`) primary, lighter grey (`#8a8ea3`) secondary
- **Typography:** System font stack only — `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`. **Do NOT use external @import or Google Fonts links** — Manifest V3's default Content Security Policy blocks remote font/script loading. If a custom font is needed, self-host the `.woff2` file inside the extension folder.
- **Shadows:** Extruded (embossed) elements use `box-shadow: 8px 8px 16px #a3b1c6, -8px -8px 16px #ffffff`. Pressed/inset elements use `box-shadow: inset 6px 6px 12px #a3b1c6, inset -6px -6px 12px #ffffff`. Light source top-left throughout.
